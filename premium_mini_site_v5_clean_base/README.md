# Premium Mini-Website V5: Niche-Aware Engine Build 🚀

A state-of-the-art, design-engineered template system for Instagram-based local businesses. Built with the **V5 Niche Engine** to ensure every site feels like a custom-coded architectural asset, not a generic template.

## 🛠 Tech Stack
- **React 19 + Vite 6**
- **Tailwind CSS v4** (Advanced @theme tokens & custom shadow-ink)
- **Motion 12** (Architectural motion sequencing)
- **Niche-Aware Engine** (Structural variance based on business DNA)

---

## 💎 V5 Engine Features
- **Archetype System:** Automatically swaps the entire page layout logic based on the niche (e.g., "Cinematic Concierge" for artists vs "Menu Commerce" for food).
- **Intent-Driven Flow:** The "Decision Dock" (mobile thumb-bar) and "Inquiry Composer" change their prompts based on the visitor's likely goal.
- **Editorial Design Language:** High visual tension, asymmetric grids, and massive fluid whitespace.
- **Android-First Mastery:** Specifically engineered for the Instagram-in-app browser experience.
- **Skill-Vendored Intelligence:** Integrated design principles from Emil Kowalski, Impeccable, and 21st.dev.

## 🚀 Creating a New Client Site (V5 Workflow)

1.  **Configure Niche:** Open `src/v5/data/v5DemoBusinesses.js`.
2.  **Select Archetype:** 
    - `cinematic-concierge`: Best for tattoo artists, makeup artists, photographers.
    - `menu-commerce`: Best for bakers, tiffin services, cafes.
    - `transformation-proof`: Best for car detailing, interior designers, salons.
3.  **Content Swap:** Update the `hero`, `offers`, and `proof` objects.
4.  **WhatsApp Sync:** Update the phone number. The engine automatically builds smart pre-filled messages based on the services.
5.  **Build & Launch:**
    ```bash
    npm run build
    ```
    Deploy the `dist/` folder to Netlify/Vercel/GitHub Pages.
